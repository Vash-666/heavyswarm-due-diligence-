## Your Task
Build a precedent transaction analysis to understand M&A valuation benchmarks:

### 1. Identify Relevant Transactions
Find 3-6 recent M&A transactions in the same industry:
- Last 3-5 years
- Similar size (within 50% of target's market cap)
- Strategic rationale alignment
- Geographic relevance

### 2. Analyze Transaction Multiples
For each transaction, extract:
- Transaction date
- Transaction value (Enterprise Value)
- EV/Revenue multiple
- EV/EBITDA multiple
- Premium paid (if public target)
- Strategic rationale

### 3. Adjust for Market Conditions
- Consider market conditions at time of deal
- Adjust for interest rate environment
- Consider strategic vs. financial buyer

### 4. Derive Implied Valuation
- Apply median transaction multiples
- Consider control premium (typically 20-30%)
- Range of values based on deal types

### 5. Assess Strategic Value
- Synergy potential
- Defensive value
- Platform value
- Technology/IP value
